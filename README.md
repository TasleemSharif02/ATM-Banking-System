# ATM Banking System

A C++ project simulating an Automated Teller Machine (ATM) banking system, developed using Visual Studio. This project demonstrates key banking operations like balance inquiry, cash withdrawal, deposits, bill payment and fund transfers in a console application.

## Abstract
An automated teller machine (ATM) is an electronic device that allows bank customers to perform financial transactions without the need for a human cashier. This project simulates the operations inside an ATM, including balance inquiries, withdrawals, deposits, bill payment and transfers, using a simple console application written in C++.

## Features
- **Balance Inquiry**: Check account balance.
- **Cash Withdrawal**: Withdraw money from your account.
- **Deposit Money**: Deposit money into your account.
- **Fund Transfer**: Transfer money to another user's account.
- **Bill Payments**: Pay bills directly from your account.
- **User-Friendly Interface**: Simple console-based user interface.
